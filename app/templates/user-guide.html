System Overview

The Oil Rig Monitoring System is a Raspberry Pi-based control and monitoring platform designed to collect sensor data and manage critical drilling equipment using GPIO-connected hardware. The system connects to the rig network via WiFi and communicates with a central operations server for data logging and remote supervision.

Safety Thresholds

  • Normal: Sensors OK.
  • Warning: Monitor closely.
  • Critical: Take emergency action immediately.

Operating Instructions

  1. Verify WiFi and server are connected.
  2. Monitor sensor readings for abnormal values.
  3. Use control buttons only if trained and authorized.
  4. Follow emergency procedures if critical thresholds are reached.

Emergency Procedure

  • 1. Activate Blowout Preventer.
  • 2. Engage Emergency Alarm.
  • 3. Shut down Mud Pump if instructed.
  • 4. Evacuate area if unsafe conditions persist.
